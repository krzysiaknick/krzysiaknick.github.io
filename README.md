## Improvements
Add English language and UTF-8 tags by modifying index.html so that it looks like this:
```
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
	<title>Page Title</title>
</head>
<body>
	<h1>This is a Heading</h1>
	<p>This is a paragraph.</p>
</body>
</html>
```
